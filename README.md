# Personal Portfolio Website 🌐

## Overview
This is a responsive Personal Portfolio Website developed as part of the Web Development Internship task. The website presents personal, academic, and professional information in a clean and modern portfolio format.

The project focuses on creating a professional online profile using modern web design principles, responsive layouts, and user-friendly navigation.

## Features
- Responsive design for mobile and laptop
- Sidebar navigation menu
- Smooth scrolling between sections
- About Me section
- Education details
- Skills section
- Internship experience
- Projects showcase
- Certifications section
- Achievements section
- Languages known
- Professional contact section
- Modern UI with popup card effects

## Technologies Used
- HTML5
- CSS3
- Font Awesome Icons
- Responsive Web Design

## Website Sections
- Home
- About Me
- Education
- Skills
- Experience
- Projects
- Certifications
- Achievements
- Languages Known
- Contact Details

## Objective
The objective of this project is to understand how AI tools and web technologies can be used to create professional websites easily while improving web development and UI design skills.

## Learning Outcomes
Through this project, I gained knowledge in:
- Responsive website development
- Website structuring using HTML
- UI styling with CSS
- Navigation design and user experience
- Portfolio website creation
- Professional web presentation
