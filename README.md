#  Student Portfolio Website

This is a simple and professional **student portfolio website** created using **HTML and CSS**.
It is designed to be used as an **online resume** and can be hosted using **GitHub Pages**.

---

##  Features
- Clean and responsive design
- Resume-style layout
- Easy to customize
- Beginner-friendly code
- Suitable for college students

---

##  Sections Included
- Name & Title
- Career Objective
- Education
- Training / Certifications
- Mini Projects
- Technical Skills
- Extra Curricular Activities
- Declaration

---

##  Technologies Used
- HTML5
- CSS3
